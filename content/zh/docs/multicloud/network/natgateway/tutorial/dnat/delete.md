    删除DNAT规则
---

该功能用于删除DNAT条目。

**单个删除**

1. 在NAT网关列表，单击NAT网关名称项，进入NAT网关详情页面。
2. 单击DNAT页签，进入DNAT页面。
2. 单击DNAT条目右侧操作列 **_"删除"_** 按钮，弹出操作确认对话框。
3. 单击 **_"确定"_** 按钮，完成操作。

**批量删除**

1. 在NAT网关列表，单击NAT网关名称项，进入NAT网关详情页面。
2. 单击DNAT页签，进入DNAT页面。
3. 在DNAT条目列表中选择一个或多个DNAT条目，单击列表上方 **_"删除"_** 按钮，弹出操作确认对话框。
4. 单击 **_"确定"_** 按钮，完成操作。
