WAF（Web Application Firewall）用于为Web应用提供集中式保护，使其免受常见攻击和漏洞的侵害。WAF可以有效识别Web业务流量的恶意特征，在对流量进行清洗和过滤后，将正常、安全的流量返回给服务器，避免网站服务器被恶意入侵导致服务器性能异常等问题，保障网站的业务安全和数据安全。


目前仅只读对接AWS、Azure、阿里云平台的WAF。
